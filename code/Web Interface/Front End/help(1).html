Help


This Help is aimed at providing the administrators of the NGO Vicharta Samuday Samarthan Manch (VSSM) with a comprehensive guide to use the NGO Information Management Suite (NIMS) in an organised manner. This Help manual will enable the administrators of VSSM to use the NIMS for visualising and analysing the collected data in an efficient way.



About NIMS


NGO Information Management Suite (NIMS) is a web-based information management system that will help VSSM in cataloguing their work in various fields in an efficient and hassle-free manner. This will aid VSSM in proper management of its resources and the field data collected with the help of a server-based system. VSSM personnel can analyze and archive data for future use.
NIMS also provides a unique feature called social mapping which enables visualization of the different field areas of VSSM and related information. Social maps will give an explicit idea to VSSM about the various ongoing and upcoming tasks in various areas and the area conditions on a visualized map.

Communities


Clicking on the Communities tab in the top navigation bar will lead you to the Communities main page shown below. This page shows a bar graph analysis of the information about communities collected by VSSM's coordinators. Clicking on a community in the list of communities shown in the left sidebar will show a statistical pie chart analysis of that particular community.


Go back to the top Visit Communities page




Projects


Clicking on the Projects tab in the top navigation bar will lead you to the Project Timeline page shown below. This page shows a timeline of the various projects undertaken by VSSM over time. Dragging the timeline to the left or right shows the projects relevant to that particular time window under focus. Clicking on a particular project's timeline will lead to a balloon pop-up displaying details about that project like is name, category, start date, end date, milestones, donor and settlements covered.


Go back to the top Visit the Projects page



Tabular Data


Clicking on the Tabular Data tab in the top navigation bar will lead you to the Tabular Data page shown below. This page shows a list of Information Tables that display the relevant data collected in the database. It also shows the District Information Table that displays all the districts in which VSSM has undertaken projects.
Clicking on a district opens the Taluka Information table for all the talukas in that district.
Clicking on a taluka in the Taluka Information table opens the Village Information table for all the villages in that taluka.
Clicking on a village in the Village Information table opens the Settlement Information table for all the settlements in that village.
Clicking on a settlement in the Settlement Information table opens the Family Information table for all the families in that settlement.
Clicking on a family in the Family Information Table opens the Member Information table for all the family members belonging to that family.
The Items Information table displays all the items distributed by VSSM that are stored in the database.
Clicking on an item opens the Item Distribution table for that item.
The Items Distribution table displays the distribution of all the items distributed by VSSM that are stored in the database.

Settings


Clicking on the Settings tab in the top navigation bar will lead you to the Settings page shown below. This page shows a list of settings available for you.
To change your password, click on the Change Password setting. It will open a form to be filled in and submitted to change your password as shown below.
To add a new coordinator to the database, click on the Add New Coordinator setting. It will open a form to be filled in and submitted to add the new coordinator as shown below.
To edit an existing coordinator's information in the database, click on the Edit existing Coordinator's Information setting. It will open a form displaying the existing coordinator's information for editing as shown below.
To add a new project to the database, click on the Add New Project setting. It will open a form to be filled in and submitted to add the new project as shown below.
To edit an existing project's information in the database, click on the Edit existing Project's Information setting. It will open a form displaying the existing project's information for editing as shown below.
To add a new village or taluka to the database, click on the Add a new Village/Taluka setting. It will open two forms for adding a new village and adding a new taluka. To add a new village, fill in and submit the Add a new village form.
To add a new taluka, fill in and submit the Add a new taluka form.
